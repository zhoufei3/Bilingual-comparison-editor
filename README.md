【使用教程】

🚀 快速开始

粘贴即翻译：在左侧粘贴原文，右侧立刻自动翻译；手动输入时，停顿约 0.6 秒后自动同步。

24 种语言：顶栏语言下拉框可自由切换源语言与目标语言，支持中、英、日、韩、法、德、西、俄等 24 种主流语言。

⇄ 互换语言：点击两个语言框之间的 ⇄ 按钮，语言方向与两侧文本内容同时互换。

⚡ 全文翻译：点击顶栏橙色「全文翻译」按钮，手动触发整段翻译，进度条逐段显示百分比。

✏️ 输入翻译模式（默认）

双向同步：左侧或右侧任意一边修改后，另一边自动翻译更新，两侧始终保持对应。

字数统计：每个面板顶部实时显示当前字符数 / 词数，方便控制内容长度。

🔍 查找替换：点击顶栏「查找」按钮展开查找替换栏，可选择在原文或译文中批量替换词汇，替换后自动触发同步翻译。

☰ 按句分行：将整段文字智能拆分为每句一行，两侧同步对齐，方便逐句对照或复制到字幕文件。再次点击「取消分行」可一键还原原始段落。

✎ 手动对照模式：开启后关闭自动翻译，两栏完全独立，可自由粘贴或输入任意内容（例如已有的两份文本手工对照）。再次点击退出，自动翻译恢复。

⇄ 对照模式

点击顶栏「对照模式」切换，文本以段落为单位左右并排展示，句子边界自动识别。

点击高亮：点击任意一侧的句子，对侧对应句子同步高亮；两侧列表同步滚动定位。

点击朗读：若「点击读句」已开启，点击句子的同时自动朗读该句的译文。

行内编辑：在对照模式中直接单击句子即可编辑，失焦后自动触发该句的精准同步翻译，无需切回输入模式。

🔍 划词翻译

底栏「划词翻译：开」按钮控制此功能的开关，状态自动保存。

在页面任意区域用鼠标选中 2 个字符以上的文本，松开鼠标后自动弹出翻译气泡，同时显示原文与译文。

气泡内：🔊 按钮朗读原文或译文（使用底栏已选发音人和语速）；☆ 按钮将该词收藏到生词本（已收藏显示 ★）。

气泡底部提供必应词典和有道词典两个链接，可直接跳转查看详细释义和例句。

气泡跟随选区自动定位，边缘碰壁时自动翻转方向；点击空白处或取消选中即关闭气泡。

⭐ 生词本

点击底栏「⭐ 生词本」按钮弹出生词本浮窗，支持拖拽标题栏移动位置，拖拽右下角调整大小。

划词翻译气泡中点击 ☆ 即可将当前词加入生词本；气泡会实时显示该词是否已收藏（★ 表示已收藏，再次点击可取消）。

列表视图：显示原文与译文，每条可独立朗读原文（🔊）或译文（🔉），或点击 ✕ 删除。

卡片视图：网格卡片展示，正面显示原文，点击卡片翻转查看译文，适合记忆练习。

底栏「⎘ 复制全部单词」将生词本内所有词以「原文 Tab 译文」格式复制到剪贴板，可直接粘贴进 Excel 或 Anki。

生词本数据包含在 JSON 备份中，导出后可在其他浏览器或设备还原。

🔊 朗读功能

点击读句：底栏「🔊 点击读句：开」开启后，在对照模式中点击任意句子，自动朗读该句译文。

朗读全文：底栏「🔊 朗读全文」按顺序逐句朗读全部译文内容；再次点击变为「⏹ 停止朗读」，随时中断。

发音人选择：底栏下拉框列出系统所有发音人，标有 ⭐ 的为 Natural / Online 高质量语音（推荐在 Edge 浏览器中使用 Microsoft Natural 语音）。划词翻译气泡的朗读按钮也使用此设置。

语速调节：发音人旁边的下拉框可选 0.5x 极慢 至 2.0x 极快，共 6 档，设置自动保存。

📖 沉浸朗读：点击顶栏「沉浸朗读」，在新标签页打开一个纯白全屏编辑器并自动填入译文内容，配合 Edge 的「大声朗读」功能（Ctrl+Shift+U）获得最佳沉浸体验。

🔖 快照与历史

存快照：点击「🔖 存快照」，输入名称后保存当前原文和译文的完整版本，可保存多个快照。

历史快照：点击「📚 历史」打开快照列表，可一键「载入」任意快照恢复内容（载入前会提示确认），也可「删除」不需要的快照。

撤销 / 重做：Ctrl+Z 撤销，Ctrl+Y 或 Ctrl+Shift+Z 重做，支持 100 步操作历史，同标签页刷新后历史不丢失。

自动保存：内容实时保存到浏览器本地存储，关闭再打开页面内容仍在，无需手动保存。

⚙️ 翻译引擎

Google 引擎（默认）：免费，无需任何配置，适合大多数场景。

Lingva：Google 翻译的免费开源镜像，作为备用引擎，网络受限时可切换尝试。

DeepL：翻译质量更高，需要在 HTML 文件顶部的 API_CONFIG 对象中填入 deepl_key。

微软翻译：同样需在 API_CONFIG 中填入 microsoft_key 和 microsoft_region。

🎨 外观与字体

主题：顶栏右侧下拉框提供 7 套主题：默认暗黑、☀️ 日间、🌊 深海蓝、🌿 森林绿、🌹 玫瑰粉、📜 暖纸米白（护眼）、🔮 紫夜，切换即时生效，下次打开自动恢复。

字体：顶栏「字体」下拉框提供多组中文字体（黑体、衬线、手写）与英文字体（衬线、无衬线、等宽）可选。

字号：点击「−」「+」按钮调整字体大小，范围 11px–26px，设置自动保存。

界面语言：顶栏最右侧「中文 / English」切换整个界面的显示语言。

💾 数据管理

点击「ℹ️ 关于」→「💾 数据管理」面板，可查看当前原文 / 译文字数、已保存快照数量、生词本词数。

导出备份：将全部内容、快照历史、生词本、主题字体等设置打包为 JSON 文件下载，换浏览器或换设备时使用。

导入还原：选择之前导出的 JSON 文件，一键恢复所有内容和设置（会覆盖当前数据，建议导入前先导出备份）。

⌨️ 快捷键一览

Ctrl+Z 撤销  ·  Ctrl+Y 重做

Esc 关闭当前弹窗（快照列表 / 关于）

Edge 沉浸朗读：Ctrl+Shift+U（需先点击「📖 沉浸朗读」打开新标签页）
